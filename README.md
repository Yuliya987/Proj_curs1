# Проект Proj_curs1

## Описание:

Проект Proj_curs1 - это веб-приложение на Python для для анализа банковских операций
Автор: Кириленко Юлия
stanislavovnajuli987@gmail.com

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/Yuliya987/Proj_curs1
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
## Использование:

1. Откройте приложение в вашем веб-браузере.
2. Введите данные и получите ответ от программы
 ### Использование разработанных функций:

1. Веб-страницы. Страница "Главная"
Функция принимает строку с датой и временем, возвращает 
по каждой карте топ-5 транзакций по сумме платежа, курс валют, 
стоимость акций

2. Сервисы. Инвесткопилка
Позволяет копить через округление ваших трат.

3. Отчеты. Траты в рабочий/выходной день
Функция выводит средние траты в рабочий и в 
выходной день за последние три месяца (от переданной даты).
